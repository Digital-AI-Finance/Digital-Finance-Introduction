## [1.0.0] - 2026-01-28

### Added
- Complete 6-day course structure covering Digital Finance
- Day 1: Why Digital Finance? (slides + NB01)
- Day 2: Platform Finance (slides + NB02-04)
- Day 3: Cryptographic Foundations (slides + NB05-07)
- Day 4: Programmable Finance (slides + NB08-10)
- Day 5: Risk and Regulation (slides + NB11-12)
- Day 6: Convergence and Future (slides + NB13-14)
- 14 hands-on Jupyter notebooks for Google Colab
- GitHub Pages website with course overview
- GitHub Actions for notebook validation and LaTeX compilation
- Comprehensive documentation (README, CONTRIBUTING, glossary)
- Learning paths for different student backgrounds
- Troubleshooting guide and FAQ

### Infrastructure
- LaTeX Beamer template for consistent slides
- Python requirements.txt for local development
- CI/CD workflows for automated builds
